# Angular app

## Installation
there are two ways to run this application:
* As node application. 
* Inside docker.

### As node application
Install node
Install typescript

Open your terminal and:
* go to /angular2-app
* run "npm start"

Open a browser and go to **localhost:3030**

### Inside docker
Install docker

Open your terminal and run:
* docker-compose up

Then open a browser and go to **localhost:8080**
